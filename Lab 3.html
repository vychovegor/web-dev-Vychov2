Opening Developer Console

To open the developer console in Mozilla Firefox, you can use the keyboard shortcut Ctrl+Shift+K or navigate to "Tools" > "Web Developer" > "Web Console". In Safari, enable the "Show Develop menu in menu bar" option in Preferences, then select "Develop" > "Show Web Inspector" and click on the "Console" tab.

Preferred Method for Including Internal Links

Generally, using relative links for internal linking within HTML pages is preferable. Relative links reference resources relative to the current document's location, offering flexibility and maintainability. Absolute links, on the other hand, specify the complete URL, which can be useful for linking to external resources or when the website structure changes.

Image Formats in Web Development

Common image formats used in web development include:

  • JPG (JPEG): Suitable for photographs and images with continuous colors.
  • GIF: Often used for animations and images with transparency.
  • PNG: Supports transparency and offers better compression without loss of quality compared to GIF.
  • WEBP: A newer image format developed by Google, providing high-quality images with efficient compression.
  • SVG: Scalable Vector Graphics, which are XML-based vector image formats supporting interactivity and scalability without loss of quality.
